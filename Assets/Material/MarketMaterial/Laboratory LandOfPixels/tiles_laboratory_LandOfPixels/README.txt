“Land of pixels” is a work in progress asset project, so my main goal is to create a lot of tilesets, sprites, UI tiles that can be use as a robust and complete tileset. And so, if you have any special request or are in need of any special asset feel free to let me know on my itch.io https://marceles.itch.io/ or as a mail on fakegamedevplus@gmail.com
